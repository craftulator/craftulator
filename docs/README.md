# Documentation

This directory contains project documentation for maintainers and agents.

## Sections

- `architecture.md`: application structure and build output.
- `ci.md`: pull request checks and branch protection behavior.
- `development.md`: local development workflow and dependency policy.
- `roadmap.md`: phased Craftulator implementation plan.
- `testing.md`: verification commands and expected checks.
- `logs/`: summaries of task discussions with users.
- `decisions/`: summaries of decisions made by agents.

All documentation in this repository must be written in English.
